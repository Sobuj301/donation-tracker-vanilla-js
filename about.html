What is the DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the page structure as a tree of objects that can be manipulated using JavaScript.

Difference between HTML and DOM?

HTML is the static structure of the page. DOM is the in-memory representation of the HTML that JavaScript can manipulate.

Difference between onclick and addEventListener?

onclick overwrites previous handlers, but addEventListener allows multiple event handlers for the same event.

Difference between parentNode and parentElement?

Both return the parent of a node, but parentElement only returns if the parent is an element.
